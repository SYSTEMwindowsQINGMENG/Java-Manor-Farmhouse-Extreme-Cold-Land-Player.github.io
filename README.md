## 📋 使用说明

1. 将代码保存为 `index.html`
2. 用浏览器打开即可访问
3. 如需替换真实二维码，将 `.qr-box` 内的模拟图形替换为 `<img>` 标签即可
4. 点击服务器地址或端口旁的 **「复制」** 按钮，可一键复制对应内容
